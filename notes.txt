Media Query:
            it is called RWD.
            Responsive web design.

            Syntax:
            @media screen and (max-width: 360px){
                /* css code for devices with width less than
            }

Example:
 <div class="main">
        <h1>Responsive web design</h1>
    </div>

Style.css
.main{
    background-color: aqua;
    height: 100px;
    font-size: 50px;
}

@media screen and (max-width: 400px){
    .main{
        background-color: red;
        height: 100px;
        font-size: 15px;
    }
}
